## Running locally

1. Clone o repo:

  ```sh
$ git clone https://github.com/Guiitzer/next-typescript-template.git
  ```

2. Vá para a pasta:

```sh
cd next-typescript-template
```

3. Instale as dependencias:

```sh
npm install 
```
*ou*
```sh
yarn install 
```

4. Rode localmente:

```sh
npm run dev
```
